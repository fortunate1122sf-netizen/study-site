目的

Spring Bootで最小のAPIを作って「Hello, JAVA!」を返す

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String sayHello() {
        return "Hello, JAVA!";
    }
}

これで得られる力

  • @RestController がどんな動きをするか体験で理解
  • @GetMapping のルーティングの仕組み
  • IntelliJやSpring Bootプロジェクトの使い方
  • curlやブラウザでの確認方法

IntelliJでSpring Bootアプリを実行する方法

  1. main() メソッドを探す（エントリーポイント）
    通常は src/main/java/〜〜〜/Application.java にある👇
@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
  1. このクラスを右クリック → Run 'Application.main()' を選択
    もしくはクラスの左横に ▶ マークが出てるので、それをクリックでもOK。
  2. コンソールに Started Application in xxx seconds と出たら成功
    この状態でAPIを叩けるようになってる。デフォルトでは http://localhost:8080/ に立ち上がってる。
  3. curlかブラウザ、またはPostmanでアクセスしてみよう
    たとえば /hello というエンドポイントがあれば
    http://localhost:8080/hello にアクセスすればレスポンスが返ってくる。
